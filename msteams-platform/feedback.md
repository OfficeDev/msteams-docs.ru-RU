# <a name="support-and-feedback"></a>Поддержка и обратная связь

## <a name="product-support-and-service-issues"></a>Поддержка продукта и проблемы с обслуживанием

Используйте **Центр администрирования Azure** или **Центр администрирования Microsoft 365** в случае любых **критически важных для бизнеса проблем**, которые требуют поддержки на основе соглашения об уровне обслуживания.

| **Поддержка продукта** | **Контакт** |
|:------------|:------------|
| **Проблемы и ошибки продукта Microsoft Teams** | Если у вас есть контракт на поддержку Premier для :::image type="icon" source="assets/icons/microsoft-icon.png"::: Microsoft 365, посетите Центр администрирования Microsoft 365 и воспользуйтесь меню поддержки, чтобы [создать запрос на обслуживание](https://admin.microsoft.com/). |
| **Помощь и поддержка по Azure** | Если у вас есть платная подписка Azure, посетите Центр администрирования Azure, чтобы [создать запрос](https://ms.portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade/newsupportreq). |
| **Общие вопросы о Teams** | Если у вас есть общие вопросы о Teams, ведите свои вопросы в :::image type="icon" source="assets/icons/microsoft-icon.png"::: [сообществе Microsoft Teams](https://answers.microsoft.com/msteams/forum).|

## <a name="developer-community-help"></a>Помощь сообщества разработчиков

Сообщество разработчиков Teams использует Stack Overflow и Microsoft Q&A для взаимодействия с другими разработчиками для идею, получения уточнений и отправки запросов. Кроме того, вы можете использовать другие контакты или сайты в зависимости от типа помощи сообщества, необходимой с возникшими проблемами, ограничениями и общими вопросами.

### <a name="developer-community-forums"></a>Форумы сообщества разработчиков

Опубликуйте свои вопросы и помогите другим участникам сообщества, делиться вопросами о разработке приложений Teams и отвечать на них.

| **Канал сообщества**|   **Контакт**  |
|:---------------------|:---------------|
|**Помощь сообщества** | <ul> <li>:::image type="icon" source="assets/icons/stack-overflow-icon.png":::[Stack Overflow](https://stackoverflow.com/questions/tagged/microsoft-teams). Используйте тег `microsoft-teams` для публикации вопросов. Следуйте рекомендациям Stack Overflow, например добавляйте описательный заголовок, краткую формулировку проблемы и детали для воспроизведения проблемы. Запросы функций или общие вопросы не рассматриваются. Если вы новый пользователь, см. раздел [Центр справки Stack Overflow](https://stackoverflow.com/help).</li>  <li>:::image type="icon" source="assets/icons/microsoft-icon.png":::[Microsoft Q&A](/answers/topics/office-teams-app-dev.html). Мы ценим Stack Overflow и предоставляем расширенную поддержку по вопросам, опубликованным на сайте Stack Overflow. Используйте тег `office-teams-app-dev` для публикации вопросов. На Stack Overflow используются критерии соответствия вопросов, и [Microsoft Q&A](/answers/topics/office-teams-app-dev.html) применяет открытую политику в отношение этого. </li><ul> |
| **Группа чата сообщества** | Используйте неформальную комнату чата сообщества для :::image type="icon" source="assets/icons/Teams-icon.png":::[разработчиков приложений Microsoft Teams](https://gitter.im/OfficeDev/MicrosoftTeamsAppDev). Выберите **SIGN IN TO START TALKING** (ВОЙТИ ДЛЯ НАЧАЛА РАЗГОВОРА), если вы входите впервые. Выберите **JOIN ROOM** (ПРИСОЕДИНИТЬСЯ К КОМНАТЕ), если вы уже вошли в систему. |

### <a name="report-issues"></a>Сообщить о проблеме

Отправляйте проблемы и задавайте общие вопросы, связанные с SDK и примерами, с помощью каналов помощи сообщества.

| **Канал сообщества** | **Контакт** |
|:----------------------|:------------|
| **Сбои платформы** | Проверьте, не рассматривалась ли ваша проблема в разделе [Сбои платформы Teams](https://aka.ms/TeamsPlatform/Outages), прежде чем создать новую. |
| **Примеры** | Отправляйте проблемы с примерами в соответствующий репозиторий [Примеры Microsoft Teams](https://github.com/OfficeDev/Microsoft-Teams-Samples).|
|  **Проблемы с набором средств Teams или TeamsFx SDK и CLI** | <ul><li> :::image type="icon" source="assets/icons/GitHub-icon.png":::  [Проблемы GitHub](https://github.com/OfficeDev/TeamsFx/issues). Создавайте новые проблемы в [репозитории GitHub для TeamsFx (Microsoft Teams Framework)](https://github.com/OfficeDev/TeamsFx), чтобы сообщать о проблемах или отправлять запросы функций. Рекомендуется использовать проблемы GitHub для запросов и получения помощи от сообщества. <li> :::image type="icon" source="assets/icons/stack-overflow-icon.png":::[Stack Overflow](https://stackoverflow.com/questions/tagged/teams-toolkit). Используйте тег `teams-toolkit` для публикации вопросов. Следуйте рекомендациям Stack Overflow, например добавляйте описательный заголовок, краткую формулировку проблемы и детали для воспроизведения проблемы. Запросы функций или общие вопросы не рассматриваются. </li> </ul> |
| **Проблемы или предложения по SDK бота и расширения для сообщений** | Отправляйте проблемы и запросы функций в репозиторий SDK для языка вашего бота ([C#](https://github.com/Microsoft/botbuilder-dotnet/), [JavaScript](https://github.com/Microsoft/botbuilder-js) или [Python](https://github.com/Microsoft/botbuilder-python)). Публикуйте вопросы о выполнении действий в [Stack Overflow](https://stackoverflow.com/questions/tagged/botframework%20microsoft-teams) с использованием тега `botframework` и `microsoft-teams`. |
| **Проблемы или предложения по SDK вкладок** | Отправляйте проблемы и запросы функций в репозиторий SDK для [библиотеки JavaScript Microsoft Teams](https://github.com/OfficeDev/microsoft-teams-library-js/issues). Используйте тег `microsoft-teams` для публикации вопросов о выполнении действий применительно к SDK в [Stack Overflow](https://stackoverflow.com/questions/tagged/microsoft-teams). |
| **Проблемы с пакетом SDK live Share или предложения** | Вы можете отправлять вопросы и запросы функций в репозиторий SDK для [Live Share SDK](https://github.com/microsoft/live-share-sdk). Используйте тег `live-share` и `microsoft-teams` для публикации практических вопросов об SDK на [Stack Overflow](https://stackoverflow.com/questions/tagged/live-share+microsoft-teams). |
