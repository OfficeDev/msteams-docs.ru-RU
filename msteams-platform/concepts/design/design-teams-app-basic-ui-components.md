# <a name="designing-your-microsoft-teams-app-with-basic-fluent-ui-components"></a>Разработка приложения Microsoft Teams с базовыми компонентами пользовательского интерфейса Fluent

Приложения teams обычно строятся с помощью следующих базовых компонентов пользовательского интерфейса Fluent. Разработанные как можно более плоские, эти компоненты могут работать в разных случаях использования, темах и размерах экрана.

На иллюстрациях на этой странице покажите, как выглядят компоненты в темах Teams по умолчанию (светлые) и темные.

## <a name="microsoft-teams-ui-kit"></a>Комплект разработчика для пользовательского интерфейса Microsoft Teams

На основе <a href="https://fluentsite.z22.web.core.windows.net/" target="_blank">пользовательского интерфейса Fluent</a>набор пользовательского интерфейса Microsoft Teams включает компоненты и шаблоны, разработанные специально для создания приложений Teams. В наборе пользовательского интерфейса можно захватить и вставить компоненты, перечисленные здесь непосредственно в дизайн, и увидеть дополнительные примеры использования каждого компонента.
